                                 CURRENCY CONVERTER
                                           
OVERVIEW:
This project is a simple Python-based command-line currency converter that converts           
an amount from one currency to another using hardcoded exchange rates. It supports 8 
major currencies and performs direct pairwise conversions.


FEATURES:
1. Supports conversion between INR,USD,GBP,EUR,JPY,CHF,CAD and AUD. 
2. Validates user input for supported currencies and positive numeric accounts.
3. Handles unsupported currency conversions gracefully with error messages.
4. Simple,interactive command-line interface.


TECHNOLOGIES/TOOLS USED:
1. Python 3.x
2. Built-in Python libraries ( no external dependencies)


STEPS TO INSTALL AND RUN THE PROJECT:
1. Ensure Python 3 is installed on your system.
2. Download or clone the project files.
3. Open a terminal/command prompt and navigate to the project directory.
4. Run the script using the command: python currency_converter.py
5. Follow the on-screen prompts to enter the souce currency, target currency and amount.


INSTRUCTIONS FOR TESTING
1. Provide valid currency codes from the supported list when prompted 
   (ex-INR,USD)
2. Enter positive numbers for the amount.
3. Try inputting an unsupported currency code to test validation.
4. Try inputting invalid or negative amounts to test error handling.
5. Confirm the output conversion matches the expected result using the hardcoded rates.
